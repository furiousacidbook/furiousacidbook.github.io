About the Book

Marijuana Manual: Technical Notebook is a practical, experiment-driven guide designed as a technical notebook—optimized for field use, iteration, and measurable outcomes.

What’s inside

  • Protocol phases (seed → veg → flower → post-harvest)
  • Environmental variables (VPD/RH/temp/wind/UV timing)
  • Lighting design notes (overhead + subcanopy strategies)
  • Failure analysis and “hidden variable” tracking

Research + Notebook Method

This work is built from iterative field trials, pilot variables, and repeatable logging. The focus is practical outcomes and reproducible protocols—not vibes (okay, a little vibes).

Field logs

Long-form observations across grow cycles; variable tracking; photo logging.

Dashboards

Target variables assigned from pilot studies → visual dashboards for decision-making.

Failure analysis

Bootstrap-style reasoning: identify hidden variables that “fasten” the model together.
